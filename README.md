# maicpp-labs
Documentação dos labs DP-300 e DP-600 - Microsoft Azure SQL e Fabric analytics

# O que é esse repositório?
Esse repositório reunira meus estudos e práticas realizadas durante o programa MAICPP. Os cursos escolhidos foram: DP-300 e DP-600.
O objetivo é documentar de forma estruturada os labs, conceitos e aprendizados adquiridos, criando um historico tecnico do meu desenvolvimento em banco de dados e analytics.

# Cursos

## DP-300 - Administering Relational Database on Microsoft Azure 
Curso focado na administração de bancos de dados relacionais no Azure, abordando configuração, monitoramento, segurança, backup, recuperação e otimização de performance. Os labs exploram cenários práticos de gestão de banco, troubleshooting e tuning, simulando atividades de um DBA em ambiente cloud.

## DP-600 - Microsoft Fabric Analytics Engineer
Curso voltado para engenharia de dados analytics utilizando Microsoft fabric, incluindo ingestão, transformação e modelagem de dados. Os labs envolvem o uso de Spark, SQL e arquiteturas de dados (medallion architecture), com foco em construção de pipelines  e analise de dados em escala
